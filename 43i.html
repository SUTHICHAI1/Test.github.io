Manual 43i Analyzer

หลักการทำงาน


       Model 43i หลักการทำงานคือโมเลกุลของ SO2 ดูดซับแสงอัลตราไวโอเลต (UV) และกระตุ้นที่ความยาวคลื่น ความยาวคลื่นหนึ่ง จากนั้นก็สลายไปอยู่ในสถานะพลังงานที่ต่ำ ซึ่งการเปล่งแสง UV มึความยาวคลื่นที่แตกต่างกัน


SO2 + hv1 (210 nm) --> SO2 *(excited Molecule) --> SO2 + hv2 (240 – 410)

       ตัวอย่างจะถูกดูดเข้า Model 43i ทางช่อง sample ดังรูป 1-1 ตัวอย่างจะไหลผ่าน hydrocarbon kicker ซึ่งกำจัด hydrocarbon ออกจากตัวอย่างโดยบังคับให้โมเลกุล hydrocarbon ซึ่มผ่านท่อพรุน แต่ถ้าเป็นโมเลกุลของ SO2 ไหลผ่าน hydrocarbon kicker จะไม่มีผลกระทบอะไร

       จากนั้นตัวอย่างจะไหลเข้าสู่ fluorescence chamber โดยที่แสง UV กระพริบเป็นจังหวะให้ไปกระตุ้นโมเลกุลของ SO2 เลนส์รวมแสงจะโฟกัสแสง UV ที่กระพริบเป็นจังหวะเข้าชุดกระจก ในชุดกระจกประกอบด้วยกระจก 4 บาน ซึ่งกระจกทั้ง 4 บานจะเลือกสะท้อนเฉพาะความยาวคลื่นที่กระตุ้นโมเลกุลของ SO2

       เมื่อโมเลกุลของ SO2 ที่ถูกกระตุ้น แล้วสลายตัวไปเป็นสถานะพลังงานที่ต่ำลง ซึ่งจะคายพลังงานออกมาในรูปแบบแสง (แสง UV) เป็นสัดส่วนกับความเข้มข้นของ SO2 ตัว Band – pass filter จะยอมให้ความยาวคลื่นที่ออกมาจาก SO2 ที่ถูกกระตุ้นเท่านั้นที่สามารถผ่านไปยังหลอด photomultiplier tube (PMT) PMT จะ detects แสงที่ได้จากการสลายตัวของ SO2

       Photodetector ที่อยู่ด้านหลัง fluorescence chamber จะค่อยตรวจสอบแสง จากแหล่งกำเนิดแสง UV ที่กระพริบเป็นจังหวะอย่างต่อเนื่องและเชื่อมต่อกับวงจรชดเชยความผันผวนของแสง UV

       เมื่อตัวอย่างไหลออกจากห้อง optical chamber ไหลผ่าน flow sensor ผ่าน capillary และ ไหลผ่านระหว่างท่อพรุนที่ hydrocarbon ซึมออกมากับผนังหนัง hydrocarbon kicker ในช่วงนี้จะนำ hydrocarbon ไหลออกไปทาง exhaust ด้วย
